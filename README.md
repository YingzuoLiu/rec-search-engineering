# Search & Recommendation Engineering Preparation

This repository is dedicated to preparing for **Search & Recommendation Platform roles** by organizing systematic practice, living-coding exercises, and end-to-end projects that cover core engineering and algorithmic challenges in large-scale retrieval and ranking systems.

## What this repository covers

- Living coding challenges focused on search & recommendation fundamentals  
- Project 1: End-to-end recommendation system prototype  
- Project 2: Search + ranking system  
- Project 3: Query understanding with LLM  
- Project 4: Multimodal cold-start solutions  
- Project 5: Evaluation & serving strategies
